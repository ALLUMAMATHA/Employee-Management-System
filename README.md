# Employee Management System

This is a *Java Swing* based Employee Management System project.  
It allows users to manage employee records with a simple GUI.

## Features
- Add Employee
- Update Employee
- View All Employees
- Delete Employee
- View Particular Employee

## Technologies Used
- Java Swing Library
- MySQL Database
- Eclipse IDE

## How to Run
1. Clone the repository or download the source code.
2. Open in Eclipse IDE.
3. Configure MySQL database with employees table.
4. Run MainFrame.java to start the application.
